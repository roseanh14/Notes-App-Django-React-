Users can sign up, log in with JWT, and manage a simple list of notes.

# Backend Setup 

From the project root:

cd backend

Run the dev server

python manage.py 

# Frontend Setup 

Open a new terminal in the project root:

cd frontend

Install dependencies

npm install

Run the dev server

npm run dev
